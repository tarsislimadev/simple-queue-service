# Simple Queue Service

In this app, clients may create simple queues.

## some rules

In index page, client clicks on create button for initialize a new Queue.

The client will be redirected to the created Queue's page.

In Queue page, client scan qrcode for managing numbers remotely.

The client scan the generated QrCode in its smartphone.

In the Control page (on client's smartphone), client may choose range of numbers, channels and more.
